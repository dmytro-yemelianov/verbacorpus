# verba — SP12: UI i18n (10 languages, URL-prefixed) — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Serve the UI chrome in 10 languages at URL-prefixed routes (`/de/…`) with server-rendered per-language `<head>` (SEO + no flash), while the corpus stays Ukrainian.

**Architecture:** A pure `i18n.ts` (langs, prefix parsing, `t()`, hreflang). The Worker becomes the front door for HTML pages: it parses the `/{lang}/` prefix and uses **HTMLRewriter** to translate `[data-i18n]` elements + head meta and inject `window.__LANG__`/hreflang per language; static assets bypass it. The client loads the active catalog for dynamically-rendered strings + a language switcher. Catalogs `i18n/{lang}.json` (uk = source); the other 8 are LLM-translated in a controller phase.

**Tech Stack:** TypeScript + Cloudflare Workers (HTMLRewriter) + esbuild + vitest; the existing PWA.

## Global Constraints

- **Languages (10):** `uk en de fr es pl it pt ja zh`; `uk` = base/source; `DEFAULT_LANG = "uk"`. No `ru`.
- **Routes:** `/` and unprefixed = uk (canonical); `/<lang>/…` = localized; `/uk/…` → **301** to the unprefixed path. Static assets (`/styles.css`, `/app.js`, `/data/*`, `/fonts/*`, `/icons/*`, `/card/*`, `/i18n/*`, `/manifest.webmanifest`, `/sw.js`, `/llms.txt`, `/robots.txt`) and `/api/*` are **never** lang-prefixed and bypass the translation layer.
- **Translate (chrome only):** masthead, search/filter labels, the 27 `cat.<key>` labels, all buttons/states, colophon, About prose, `/p/:id` chrome, OG/Twitter meta, api.html docs. **Never:** proverb `text`/`modern_text`/`explanation`, source names, the `/card` images, JSON data, API bodies, `llms.txt`.
- **Catalogs:** `app/public/i18n/<lang>.json`, flat `key→string`; `uk.json` is the source-of-truth (every catalog has the identical key set). Bundled into the Worker (JSON import) AND served as assets.
- **No FOUC:** static `[data-i18n]` chrome is translated **server-side** by the Worker; the client catalog covers only dynamically-rendered strings (results/swipe/saved/counts/category labels) and loads before first dynamic render.
- **SEO:** each HTML page gets `<html lang>`, a per-language `<link rel="canonical">`, `<link rel="alternate" hreflang>` for all 10 + `x-default`, and `og:locale`.

- **Build order:** engine + uk + **en** first (prove end-to-end); the other 8 are a controller phase.

- [ ] **Step 3: Implement** — `app/src/shared/i18n.ts`:
```typescript
export const LANGS = ["uk", "en", "de", "fr", "es", "pl", "it", "pt", "ja", "zh"];
export const DEFAULT_LANG = "uk";
export const LANG_NAMES: Record<string, string> = {
  uk: "Українська", en: "English", de: "Deutsch", fr: "Français", es: "Español",
  pl: "Polski", it: "Italiano", pt: "Português", ja: "日本語", zh: "中文",
};

export function parseLang(pathname: string): { lang: string; rest: string } {
  const m = pathname.match(/^\/([a-z]{2})(\/.*|$)/);
  if (m && LANGS.includes(m[1]) && m[1] !== DEFAULT_LANG) {
    return { lang: m[1], rest: m[2] === "" ? "/" : m[2] };
  }
  // /uk or /uk/... → treat as uk with the rest (caller redirects to canonical)
  if (m && m[1] === DEFAULT_LANG) return { lang: "uk", rest: m[2] === "" ? "/" : m[2] };
  return { lang: DEFAULT_LANG, rest: pathname };
}

export function t(catalog: Record<string, string>, key: string, fallback?: string): string {
  return catalog[key] ?? fallback ?? key;
}

export function hreflangLinks(restPath: string, host: string): string {
  const p = restPath === "/" ? "" : restPath;
  const href = (lang: string) => `https://${host}${lang === DEFAULT_LANG ? "" : "/" + lang}${p || (lang === DEFAULT_LANG ? "/" : "")}`;
  const links = LANGS.map((l) => `<link rel="alternate" hreflang="${l}" href="${href(l)}" />`);
  links.push(`<link rel="alternate" hreflang="x-default" href="${href(DEFAULT_LANG)}" />`);
  return links.join("\n");
}
```

```typescript
function translateHtml(res: Response, lang: string, rest: string, host: string): Response {
  const cat = catalogFor(lang);
  const inject =
    `<script>window.__LANG__=${JSON.stringify(lang)}</script>\n` +
    hreflangLinks(rest, host) + `\n` +
    `<link rel="canonical" href="https://${host}${lang === DEFAULT_LANG ? "" : "/" + lang}${rest === "/" ? "/" : rest}" />`;
  return new HTMLRewriter()
    .on("html", { element(el) { el.setAttribute("lang", lang); } })
    .on("[data-i18n]", { element(el) { const k = el.getAttribute("data-i18n"); if (k) el.setInnerContent(t(cat, k), { html: false }); } })
    .on("[data-i18n-attr]", { element(el) {
        const spec = el.getAttribute("data-i18n-attr") || "";
        const [attr, key] = spec.split(":");
        if (attr && key) el.setAttribute(attr, t(cat, key));
      } })
    .on("head", { element(el) { el.append(inject, { html: true }); } })
    .transform(res);
}
```
